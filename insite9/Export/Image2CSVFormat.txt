"This document details the output format of INSITE's ""Export Image to external file"" functionality."									
"""Export Image to external file"" produces similarly formatted output as a printed ECM Image. "									
The most significant difference is a comma-delimited file is created instead of printer output									
The output file is stored in the user-specified directory (the default location is the directory in Tools-> Options-> File -> Export).									

The following tables detail the organization of data contained within an external file:									
NOTE: Some tables will not be present depending upon which features are available within the ECM. 									
NOTE: Placement of blank lines in an image output may differ than those within this document.
